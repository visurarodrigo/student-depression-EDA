# Student Depression Exploratory Data Analysis (EDA)

## Project Overview
This project presents an exploratory data analysis of student depression among Grade 12 students using R.
The analysis focuses on identifying the major academic, lifestyle, and socioeconomic factors associated with depression risk.

Dataset source: [Kaggle - Student Depression Dataset](https://www.kaggle.com/datasets/adilshamim8/student-depression-dataset)

## Objectives
- Identify the factors most strongly associated with student depression.
- Summarize key variables using descriptive statistics.
- Visualize meaningful patterns and relationships in the data.
- Generate evidence-based insights that can support early intervention.

## Repository Structure
- `EDA_Report.pdf` - Full analytical report with interpretation and discussion.
- `eda_script.R` - R script used for data exploration and visualization.
- `README.md` - Project documentation.

Note: The dataset file (`student_depression_dataset.csv`) can be downloaded from Kaggle and placed in the project root before running the script.

## Tools and Technologies
- R
- Base R plotting

## Methodology Summary
1. Data loading and inspection.
2. Data cleaning and preparation.
3. Univariate and bivariate exploratory analysis.
4. Visual analysis of key trends (for example, sleep, academic pressure, diet, and financial stress).
5. Interpretation of findings in the context of student mental health.

## Key Findings
- Academic pressure and sleep patterns show strong relationships with depression levels.
- Female students exhibit slightly higher depression rates in this dataset.
- Financial stress and poor dietary patterns are notable contributing factors.

## How to Run
1. Install R on your machine.
2. Download `student_depression_dataset.csv` from Kaggle.
3. Place the dataset in the project root directory.
4. Open and run `eda_script.R` in RStudio or any R environment.
